# 關東東京五日自助旅行指南 (kanto-travel-guide)

這是一個專為日本東京/關東五日自助旅行設計的單頁式互動旅遊指南。本指南特別融入了精美的日式視覺風格，並針對行動裝置、桌面端及紙張列印進行了優化，非常適合分享給家人或列印成實體紙張給長輩參考。

## 專案特點

- **日式和風美學**：採用朱紅、墨黑、和紙色等經典日式配色，呈現極具質感的視覺設計。
- **互動式行前清單**：提供可互動勾選的行前準備檢查表，方便登機前或打包時快速確認。
- **每日行程導航**：使用標籤頁（Tabs）切換 Day 1 至 Day 5 的詳細交通指引，點擊即可流暢切換。
- **數據圖表化**：整合 Chart.js，以折線圖和長條圖直觀展示「每日體力消耗預測」與「每日交通花費與明細」。
- **專業列印樣式**：已加入 `@media print` 專屬樣式。列印時會自動隱藏導航列、按鈕等互動元素，自動展開所有折疊細節與每日行程，並將字體縮小與去底色，完美契合 A4 紙張列印（省墨水且好閱讀）。

## 技術架構

- **前端結構**：HTML5 語意化標籤
- **樣式設計**：Tailwind CSS (透過 CDN 載入)
- **圖表繪製**：Chart.js (透過 CDN 載入)
- **互動邏輯**：原生 JavaScript (Vanilla JS)

## 更新與維護流程

若未來需要修改或更新旅遊行程：
1. 修改本專案的 `index.html` 檔案內容。
2. 將修改後的檔案 commit 並 push 至 GitHub 上的 `main` 分支。
3. GitHub Actions 會自動偵測並重新部署，約 1-2 分鐘後公開網址即會自動更新。
