Before departure

行前準備

出發前先確認證件、支付、交通卡與日本旅遊習慣。這一區就是最後打包與登機前的快速檢查表。

必備物品

Suica 交通卡

抵達成田機場第二航廈後先處理交通卡與儲值，建議一開始儲值 5000 日圓。後續搭電車、便利商店小額支付都會順很多。

Train notes

電車車種筆記

最常踩雷：搭錯車種。東京搭車時，確認列車終點方向與停靠站比只看路線名更重要，尤其遇到直通運轉與不同停靠等級時。

最常踩雷：同一月台可能來各停、急行、特急，停靠站不同，先確認車種再上車。
判斷重點：確認列車終點方向與停靠站，避免搭到方向相同但終點不同的列車。
車種顏色特性提醒
各停灰每站停靠最穩但較慢
急行藍跳過部分站先確認目的站有停
特急橘主要大站停靠部分列車需加購

Daily guide

每日詳細交通指引

Arrival

Day 1｜8/29 成田抵達與下町暖身

  1. 06:35 虎航 IT200 台灣出發。
  2. 11:00 抵達成田機場第二航廈，先處理 Suica / 交通卡與儲值。
  3. 午後 搭車前往 Sumida Base，放行李後到押上或淺草熟悉周邊。
  4. 晚餐 淺草周邊輕鬆吃；若想熱鬧一點，可排鳥貴族淺草六區店。
實用日語

請問往押上的車在哪裡？可說「押上方面はどちらですか」。買票、問路時把站名給站務員看最穩。

天氣對策

抵達日先把摺疊傘放在隨身包，午後若遇雷雨，押上與淺草都有室內商場或店家可躲雨。

Asakusa / Skytree

Day 2｜8/30 淺草、上野、晴空塔

  1. 上午 淺草寺、仲見世通、雷門，避開正午人潮先拍照。
  2. 中午 上野公園或阿美橫町散步採買，小吃與藥妝可先補一輪。
  3. 下午 回押上逛東京 Solamachi，預留時間上晴空塔周邊拍照。
  4. 17:30 敘敘苑 東京晴空塔城 Solamachi 30F，2 位訂位。
  5. 18:00 敘敘苑 東京晴空塔城 Solamachi 30F，4 位訂位。
實用日語

餐廳報到可說「予約しています」，再出示訂位姓名與人數。分成 17:30 兩位、18:00 四位要特別提醒。

天氣對策

淺草到上野多為戶外步行，若太熱就把上野縮短，提早進 Solamachi 室內逛街休息。

Fuji Day

Day 3｜8/31 河口湖、富士山包車

  1. 清晨 從 Sumida Base 出發，建議早餐與水先備好。
  2. 上午 河口湖、大石公園或富士山景觀點，依天氣調整拍照順序。
  3. 午後 可加入忍野八海、新倉山淺間公園或富士山周邊咖啡停留。
  4. 晚餐 回東京後保留彈性；若還有精神，可補大福園街頭燒肉。
實用日語

包車集合時可用「おはようございます、よろしくお願いします」。臨時改點可說「天気を見て順番を変えたいです」。

天氣對策

富士山能見度很吃天氣。若雲厚，先排近景、湖邊與咖啡店，看到山頭露出再衝拍照點。

Shibuya Sunset

Day 4｜9/1 原宿、表參道、澀谷夕陽

  1. 上午 明治神宮、原宿竹下通，城市與綠意先交錯開場。
  2. 午後 表參道、青山一帶散步，安排咖啡休息避免步行過載。
  3. 傍晚 Shibuya Sky 高空夕陽，建議預約夕陽前 60-90 分鐘入場時段。
  4. 晚餐 澀谷周邊用餐或回淺草吃鳥貴族；本日步行量最大，確認列車終點方向與停靠站。
實用日語

Shibuya Sky 入場可準備電子票。詢問排隊入口時可說「入口はどちらですか」。

天氣對策

高空展望遇強風或雨勢可能影響戶外區域。若天氣差，改把澀谷購物與餐廳時間拉長。

Departure

Day 5｜9/2 採購及返程

  1. 上午 退房與行李整理；可去押上 Solamachi、淺草或上野做最後採購。
  2. 中午 建議 12:30 前後往成田機場第二航廈移動，保留報到與購物時間。
  3. 16:20 星宇 JX803 起飛。
  4. 19:00 抵達台灣，結束五日東京旅・記。
實用日語

寄物或詢問置物櫃可說「コインロッカーはありますか」。機場報到出示護照與電子機票即可。

天氣對策

返程日不要排太遠的戶外點。若下雨，押上 Solamachi 或上野車站周邊採購最省力。
